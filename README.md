## Getting Started

1. Fork this repo
2. Make a branch like `git checkout -b my_name/my_ai_name`
3. Run `bundle exec thor new_ai my_ai_name` to get a new ai
4. Test it locally with `bin/runner`
5. Commit it and make a pull request
